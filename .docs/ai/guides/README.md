# 03_tool_guides

AI コーディングツールの「使いこなし」に関するガイド集です。

## このディレクトリの位置づけ

- **読み手**: 人間（AI に渡すコンテキストではない）
- **目的**: ツールを効率的に使うための知識・ベストプラクティス
- **特徴**: 特定ツールに依存しない汎用知識

## ファイル一覧

|----------|------|
| `tool_search.md` | Tool Search Tool の解説と活用判断 |
