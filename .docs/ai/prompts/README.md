# 03_prompts

AI への作業依頼テンプレートをまとめたディレクトリです。

## 目的

- **作業依頼の標準化**: よく使う指示をテンプレート化
- **効率的な AI との対話**: コピペで使えるプロンプト集

## 使い方

作業中に、該当するプロンプトテンプレートをコピーして AI に貼り付けます。

例：コードのモジュール化を依頼する時
```text
「03_prompts/コードのモジュール化の指示.md のステップ1を実行してください」
```

## 構成

- `コードのモジュール化の指示.md` - リファクタリング用の3ステップ指示
- `llm_prompts.md` - その他の LLM 用プロンプト
- `planning/` - 計画系プロンプト（KPT、Issue作成など）

## このディレクトリに置くもの / 置かないもの

### 置くもの

- AI への作業依頼テンプレート
- コピペして使うプロンプト

### 置かないもの

- 思想・心構え → `01_agent_context/`
- コードスニペット → `04_code_snippets/`
