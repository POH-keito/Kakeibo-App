# 02_tool_specs

特定のツールやサービスを使用する際に、AI に読ませる仕様書・ドキュメントをまとめたディレクトリです。

## 目的

- **ツール固有の知識を共有**: 特定ツールを使う時に AI に仕様を理解させる
- **一貫した実装**: ツールのベストプラクティスに沿った実装を促す

## 使い方

特定のツールを使うプロジェクトで、該当する仕様書を AI に読み込ませてください。

```text
「02_tool_specs/BrowserAct/ 配下のドキュメントを読み込んでください。
この仕様に基づいて実装を進めます。」
```

## 構成

- `BrowserAct/` - BrowserAct の仕様書
- `NocodeBackend/` - NoCodeBackend の仕様書

## このディレクトリに置くもの / 置かないもの

### 置くもの

- ツールやサービスの仕様書（AI に読ませる用）
- ツール固有のベストプラクティス

### 置かないもの

- 一般的な開発思想 → `01_agent_context/`
- コードスニペット → `04_code_snippets/`
