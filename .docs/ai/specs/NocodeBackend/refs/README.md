## 2. 情報の取り扱い方針

SpecSheet に情報を反映する際は、以下のルールに従ってください。

| 項目               | 方針                                                                                        |
| :----------------- | :------------------------------------------------------------------------------------------ |
| **情報源の優先度** | 質問者やレビュー投稿者の意見よりも、**開発者（回答者）の回答を正**とします。                |
| **情報の競合**     | 同じ内容について複数の情報がある場合は、**より新しい情報を優先**します。                    |
| **目的**           | これらの情報を基に、AI が利用する上で有用な仕様、ベストプラクティス、制約などを抽出します。 |
