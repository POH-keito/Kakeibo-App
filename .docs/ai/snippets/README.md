# 04_code_snippets

コピペして使えるコードスニペットや定型文をまとめたディレクトリです。

## 目的

- **再利用可能なコード**: よく使うコードパターンを保存
- **定型文の管理**: Issue や PR 作成時に使う定型文

## 使い方

コードを書く時や文章を書く時に、該当するスニペットをコピーして使用します。

## 構成

- `GitHub_Make_PR.md` - PR 作成時の定型文
- `ime_support_request.md` - IME 対応のコードスニペット
- `Simple_LLM_Output.md` - LLM 出力の簡易フォーマット

## このディレクトリに置くもの / 置かないもの

### 置くもの

- コピペして使うコードスニペット
- 定型文・テンプレート

### 置かないもの

- AI への指示テンプレート → `03_prompts/`
- 学習メモ → `05_knowledge/`
