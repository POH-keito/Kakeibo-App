# 01_agent_context

AI コーディングエージェントに**セッション開始時に最初に読ませる**思想・心構えをまとめたディレクトリです。

## 目的

- **Vibe Coding の思想を共有**: 開発スタイルの基盤となる考え方を AI に理解させる
- **一貫した開発体験**: どのプロジェクトでも同じ思想で AI と協働できる
- **初期コンテキストの確立**: セッション開始時に「前提」を揃える

## 使い方

新しいプロジェクトやセッション開始時に、このディレクトリのドキュメントを AI に読み込ませてください。

```text
「01_agent_context/vibe_coding/ 配下のドキュメントを読み込んでください。
この構成に基づいて開発を進めます。」
```

## 構成

- `vibe_coding/` - Vibe Coding の思想・技術スタック・開発フローなど（詳細は [vibe_coding/00_README.md](vibe_coding/00_README.md) 参照）
- `mcp_docker_workflow.md` - MCP サーバー利用時の AI への初期指示

## このディレクトリに置くもの / 置かないもの

### 置くもの

- AI に「思想・心構え」として読ませるドキュメント
- 開発の方針や原則を定めるもの
- セッション開始時に共有すべき前提知識

### 置かないもの

- 技術的なコードスニペット → `04_code_snippets/`
- AI への作業依頼テンプレート → `03_prompts/`
- 特定ツールの設定手順 → `02_tool_specs/`
